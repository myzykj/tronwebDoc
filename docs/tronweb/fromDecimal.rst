
fromDecimal
=============

Converts a number, or a string of numbers, into a hexadecimal string.

-------
Usage
-------

.. code-block:: javascript
  
  tronWeb.fromDecimal()

--------------
Parameters
--------------

Number | String

-------
Returns
-------

String

-------
Example
-------

.. code-block:: javascript

  tronWeb.fromDecimal("21")
  > "0x15"
