
fromSun
=============

Helper function that will convert a value in SUN to TRX. (1 SUN = 0.000001 TRX)

-------
Usage
-------

.. code-block:: javascript
  
  tronWeb.fromSun()

--------------
Parameters
--------------

Number | String

-------
Returns
-------

String

-------
Example
-------

.. code-block:: javascript

  tronWeb.fromSun("1000000")
  > '1'
