
fromUtf8
=============

Helper function that will convert UTF8 to HEX

-------
Usage
-------

.. code-block:: javascript
  
  tronWeb.fromUtf8()

--------------
Parameters
--------------

String

-------
Returns
-------

String

-------
Example
-------

.. code-block:: javascript

  tronWeb.fromUtf8("test")
  >"0x74657374"
