isAddress
==========

Helper function that will check if a given address is valid.

-------
Usage
-------

.. code-block:: javascript
  
  tronWeb.isAddress()

--------------
Parameters
--------------

String

-------
Returns
-------

Boolean

-------
Example
-------

.. code-block:: javascript

  tronWeb.isAddress("414fa1f834a47f621957ec2ae7d445da9b3be0bee4")
  >true
  tronWeb.isAddress("THEGR4Aor5pCDVktbbbwgHAE6PQWRfejBf")
  >true
