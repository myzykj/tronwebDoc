toAscii
==========

Convert HEX string to ASCII3 string

-------
Usage
-------

.. code-block:: javascript
  
  tronWeb.toAscii()

--------------
Parameters
--------------

String

-------
Returns
-------

String

-------
Example
-------

.. code-block:: javascript

  tronWeb.toAscii("0x74726f6e")
  >"tron"
