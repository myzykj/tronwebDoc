toDecimal
==========

Convert a hexadecimal to a decimal number

-------
Usage
-------

.. code-block:: javascript
  
  tronWeb.toDecimal()

--------------
Parameters
--------------

String - Hex string

-------
Returns
-------

Number - The hexadecimal value represented by the passed in string.

-------
Example
-------

.. code-block:: javascript

  tronWeb.toDecimal('0x15')
  >21
