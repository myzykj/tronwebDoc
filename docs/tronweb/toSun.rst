toSun
=======

Helper function that will convert a value in TRX to SUN. (1 SUN = 0.000001 TRX)

-------
Usage
-------

.. code-block:: javascript
  
  tronWeb.toSun()

--------------
Parameters
--------------

Number

-------
Returns
-------

String

-------
Example
-------

.. code-block:: javascript

  tronWeb.toSun(10)
  >"10000000"
