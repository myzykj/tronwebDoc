timeUntilNextVoteCycle
===========

Query the time interval till the next vote round

-------
Usage
-------

.. code-block:: javascript

  tronWeb.trx.timeUntilNextVoteCycle();

--------------
Parameters
--------------

N/A

-------
Returns
-------

number

-------
Example
-------

.. code-block:: javascript

  >tronWeb.trx.timeUntilNextVoteCycle().then(console.log)
  >1579068000
